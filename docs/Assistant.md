# Smart Assistant

The Smart Assistant is a conversational AI interface within OSSI Tools that allows users to query BlueYonder WMS data using plain, everyday language. Simply type a question into the message box to instantly retrieve information and uncover insights from your warehouse data no technical expertise or complex navigation required.
