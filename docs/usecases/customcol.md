# Custom Column Maintenance 

By default, custom columns added to tables are not visible on the 
screen. To view and manage these custom columns, **Smart VIU** provides a report called Custom Column Maintenance.

This report allows you to look up custom columns along with the standard columns for any table and perform 
full CRUD operations directly from the 
interface.



## Accessing Custom Column Maintenance

To get started, navigate to the **Custom Column Maintenance** report in Smart VIU.

## Viewing Custom Columns

To view the custom columns for a table:

1. Open the Custom Column Maintenance report.
2. Enter the **table name** in the provided field and click 'find'.

<img src="/docs/Attachments/customcol2.png" alt="undirectedmenu" style="height: 250px; width:500px;margin:auto;display:block; cursor: zoom-in;
