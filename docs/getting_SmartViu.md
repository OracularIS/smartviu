# Getting Started with SmartViu

## System Requirements
To use SmartViu, make sure your system meets the following requirements:
-	Web browser (latest version recommended)
-	Internet access
-	Blue Yonder WMS credentials

## Accessing SmartViu
-	Open your preferred web browser.
-	Navigate to the BY Portal and login using your credentials.
-	Go to OSSI Tools Menu
-	Click on SmartViu menu.
