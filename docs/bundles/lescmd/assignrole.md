# Assign Role 

Follow these steps to assign role assignment from a specific LES command:

1. Click on the "Find" button to load the LES command record in the grid.
2. Select a record from the grid.
3. Click on the "Role ID" button to assign the roles for the selected LES command.
4. Now, the role for the selected LES command is added.
